管理员 / 应用发布台

应用发布台

管理员发布的是应用定义和运行方式，不是某一个固定图文模板。提示词、工作流、智能体和外部托管应用都走版本化发布，用户运行时再选择自己的平台 API 密钥。

可发布形态与开发方式

提示词应用
轻量
适合翻译、摘要、标题生成、简单文本处理，也可以带文件输入。
工作流应用
节点
适合固定流程：文件解析、检索、模型调用、分支判断、产物导出。
智能体应用
工具
配置目标、系统提示词、工具权限、最大步数和失败兜底策略。
外部托管应用
Worker
复杂生视频、批量处理和自有工作流引擎通过 Webhook/Worker 接入。
管理员主要做配置式开发：选择已有节点、工具和外部服务，配置输入输出、模型角色和权限后发布。需要写代码的能力由开发者做成外部 Worker 或平台内置节点，再交给管理员接入。

管理员 / Worker Host

Worker Host

管理员在这里维护可复用的 Worker 服务地址。应用版本只绑定 Host 和运行路径，Sub2API 负责健康检查、签名调用、并发控制和回调接收。

Host 配置

不绑定域名也可以使用 IP:端口，但必须限制防火墙，只允许 Sub2API 主服务器访问，并启用签名鉴权。
